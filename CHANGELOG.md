# 更新日志

## 未发布

### 修复

- 修复聊天页在部分自定义模型或第三方模型厂商下点击“续写”后可能没有内容的问题。
- 优化续写提示的模型兼容性，默认使用更通用的用户消息形式触发续写，并保留运行时配置入口，方便特殊模型按需调整。
- 优化续写/重新生成的空结果处理：如果模型没有返回有效内容，后端会明确标记失败并返回错误提示，避免出现请求成功但聊天框无内容的情况。
- 保留用户主动停止生成时的原有行为，避免把正常取消误判为模型失败。

### 影响范围

- H5 聊天页：续写、重新生成。
- 后端生成审计：空结果不再误标为成功。
- SillyTavern runtime 集成：续写提示角色与提示内容支持配置化。
