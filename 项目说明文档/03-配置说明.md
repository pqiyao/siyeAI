# 配置说明

项目主要通过环境变量配置。请复制 `.env.example` 或 `deploy/.env.example` 后再修改，不要把真实 `.env`、密钥、证书或生产配置提交到仓库。

## 配置文件入口

| 位置 | 用途 |
| --- | --- |
| `.env.example` | 根目录环境变量样例，适合理解后端和本地开发配置。 |
| `deploy/.env.example` | Docker Compose 部署环境变量样例。 |
| `backend/src/main/resources/application.yaml` | 后端 Spring Boot 主配置，绝大多数值通过环境变量注入。 |
| `admin-web/.env.development.example` | 管理端开发环境样例。 |
| `admin-web/.env.production.example` | 管理端生产构建样例。 |
| `h5-web/common/api.js` | H5 API、图片、上传、WebSocket 地址选择逻辑。 |

## 根目录 `.env.example`

常用变量：

| 变量 | 说明 |
| --- | --- |
| `MYSQL_ROOT_PASSWORD` | MySQL root 密码，Docker 场景使用。 |
| `MYSQL_DATABASE` | 默认数据库名。 |
| `MYSQL_USER` | 应用数据库用户。 |
| `MYSQL_PASSWORD` | 应用数据库密码。 |
| `SPRING_DATASOURCE_URL` | 后端 JDBC 连接串。 |
| `SPRING_DATASOURCE_USERNAME` | 后端数据库用户名。 |
| `SPRING_DATASOURCE_PASSWORD` | 后端数据库密码。 |
| `SPRING_DATA_REDIS_HOST` | Redis 地址。 |
| `SPRING_DATA_REDIS_PORT` | Redis 端口。 |
| `SPRING_DATA_REDIS_PASSWORD` | Redis 密码，可为空。 |
| `APP_AUTH_SECRET` | H5/API 用户 token 签名密钥。 |
| `APP_RUOYI_ADMIN_USERNAME` | 管理端初始账号。 |
| `APP_RUOYI_ADMIN_PASSWORD` | 管理端初始密码的 bcrypt 值。 |
| `APP_RUOYI_JWT_SECRET` | 管理端 JWT 密钥。 |
| `APP_CORS_ALLOWED_ORIGIN_PATTERNS` | 允许跨域来源。 |
| `SILLYTAVERN_BASE_URL` | 后端访问 SillyTavern 的服务端地址。 |
| `SILLYTAVERN_PUBLIC_BASE_URL` | 返回给前端使用的 SillyTavern 公共地址。 |
| `SILLYTAVERN_API_KEY` | SillyTavern API Key，可按需启用。 |
| `TELEGRAM_BOT_TOKEN` | Telegram 登录或 Stars 支付相关配置。 |

## 后端配置分组

后端主配置位于：

```text
backend/src/main/resources/application.yaml
```

主要分组：

| 分组 | 说明 |
| --- | --- |
| `app.auth` | 用户认证 token 配置。 |
| `app.ruoyi-admin` | 管理端账号、JWT、验证码等配置。 |
| `app.chat` | 聊天并发、队列、SSE 超时和生成超时。 |
| `app.memory` | 对话记忆摘要和 LLM 记忆开关。 |
| `app.image-generation` | 图像生成引擎、并发、Comfy/OpenAI-compatible 配置。 |
| `app.api-rate-limit` | API 全局限流。 |
| `app.social-upload-rate-limit` | 上传、社区写入、WebSocket 握手限流。 |
| `app.websocket` | WebSocket Origin 和握手 token 策略。 |
| `app.payment` | 模拟支付、微信 H5、支付宝 WAP、Telegram Stars 配置。 |
| `telegram` | Telegram bot 和登录校验相关配置。 |
| `sillytavern` | SillyTavern 地址、API Key、模型、反代和调试配置。 |
| `spring.datasource` | MySQL 连接。 |
| `spring.data.redis` | Redis 连接。 |
| `spring.flyway` | Flyway 迁移。 |
| `mybatis` | MyBatis XML mapper。 |
| `management` | Actuator 健康检查和指标。 |

## H5 配置

H5 主要运行配置位于：

```text
h5-web/common/api.js
```

常见配置含义：

- `LOCAL_API_ORIGIN`：本地浏览器开发时的 API 地址。
- `APP_API_ORIGIN`：App/Html5Plus 运行时 API 地址。
- `REMOTE_API_ORIGIN`：远程 H5 浏览器部署时 API 地址。
- `REMOTE_IMG_ORIGIN`：图片资源域名。
- `REMOTE_UPLOAD_ORIGIN`：上传资源域名。
- `REMOTE_SOCKET_URL`：WebSocket 地址。

生产环境建议通过同源反向代理访问后端，减少 CORS 和 WebSocket Origin 配置复杂度。

## SillyTavern 配置

后端通过 `sillytavern.*` 配置访问 SillyTavern。

| 环境变量 | 说明 |
| --- | --- |
| `SILLYTAVERN_BASE_URL` | 后端调用 SillyTavern 的服务端地址。Docker 中通常不能写容器内不可达的本机地址。 |
| `SILLYTAVERN_PUBLIC_BASE_URL` | 返回给前端使用的公共地址。 |
| `SILLYTAVERN_API_KEY` | SillyTavern API Key。 |
| `SILLYTAVERN_DEFAULT_MODEL` | 默认模型。 |
| `SILLYTAVERN_CHAT_COMPLETION_SOURCE` | 默认补全来源。 |
| `SILLYTAVERN_ASSET_CACHE_DIR` | ST 资源缓存目录。 |
| `SILLYTAVERN_DEBUG_ENABLED` | 调试接口开关。 |

如果后端运行在 Docker 中，请确认容器内能访问 `SILLYTAVERN_BASE_URL`，而不是只有宿主机浏览器能访问。

## 安全建议

- 所有 `change-this-*` 都必须替换为强随机值。
- `APP_AUTH_SECRET` 和 `APP_RUOYI_JWT_SECRET` 建议至少 32 字节随机字符串。
- 管理端密码使用 bcrypt 存储，格式示例为 `{bcrypt}...`。
- 不要在前端代码中写入生产密钥。
- 生产环境关闭不需要的 debug 能力。
- 支付私钥建议通过环境变量或安全挂载注入，不要提交到仓库。
- 如果任何真实密钥曾经进入仓库，发布前应立即轮换。
