# 前端说明

仓库包含两个前端项目：管理后台 `admin-web/` 和 H5 客户端 `h5-web/`。二者都通过后端 API 工作，但面向的用户和构建方式不同。

## 管理后台 `admin-web`

`admin-web/` 是 Vue 3 + Vite + Element Plus 的运营管理端，整体风格基于 RuoYi Vue 后台。

### 启动

```bash
cd admin-web
npm install
npm run dev
```

构建：

```bash
npm run build:prod
```

预览：

```bash
npm run preview
```

### 常见目录

```text
admin-web/
  src/
    api/            API 请求封装
    assets/         静态资源
    components/     通用组件
    layout/         后台布局
    router/         路由
    store/          Pinia 状态
    utils/          request、权限、主题等工具
    views/
      jiugai/       业务运营页面
      login.vue     登录页
      index.vue     首页
```

## H5 客户端 `h5-web`

`h5-web/` 是 uni-app H5 客户端，覆盖发现、酒馆、聊天、会话、用户中心、社区、私聊、工单、支付、设置等用户侧流程。

### 安装依赖

```bash
cd h5-web
npm install
```

然后使用 HBuilderX 或兼容的 uni-app 工具链运行或构建 H5。

### 页面结构

主要页面位于 `h5-web/pages/`。

| 目录 | 用途 |
| --- | --- |
| `index` | 发现页和搜索。 |
| `login` | 登录、注册、找回。 |
| `tavern` | 角色列表、详情、编辑器、会话、聊天、人设。 |
| `chat` | 系统消息。 |
| `social` | 社区动态、发布、详情、私聊列表、私聊页。 |
| `user` | 用户中心、VIP、钱包、收藏、设置、AI 设置、Live2D、游戏、工单、协议。 |
| `perfect` | 信息完善流程。 |
| `tuikuan` | 退款说明页面。 |

### H5 API 封装

核心请求配置：

```text
h5-web/common/api.js
```

酒馆、角色、聊天、社区、工单等业务封装：

```text
h5-web/common/tavernApi.js
```

聊天相关辅助：

```text
h5-web/common/chatMarkdown.js
h5-web/common/tavernPrompt.js
h5-web/common/tavernErrors.js
h5-web/common/tavernCompliance.js
```

### 多语言

语言文本位于：

```text
h5-web/common/text/
```

当前包含 `zh-cn`、`zh-hk`、`en`、`ja`、`ko`、`fr`、`de` 等 JSON 文件。

### 静态资源

H5 静态资源位于：

```text
h5-web/static/
```

包括 tabbar、用户中心、游戏、工作流、图片、音频和其他前端资源。

发布前请确认资源版权和体积，尤其是图片、音频、Live2D、游戏素材。

## 前后端联调

- 管理端默认通过 Vite 代理请求后端。
- H5 本地浏览器默认使用 `http://127.0.0.1:8080` 或 `common/api.js` 中配置的地址。
- 生产建议 H5 与后端走同源反向代理。
- 如果出现跨域问题，检查后端 `APP_CORS_ALLOWED_ORIGIN_PATTERNS` 和 WebSocket Origin 配置。
