# 接口地图

本文是按功能整理的接口入口地图，方便二次开发时快速定位 Controller、前端调用和业务模块。它不是完整 OpenAPI 文档，具体请求字段、返回字段和校验规则请以对应 Controller、DTO、Service 和前端 API 封装为准。

## H5 / App 用户认证

Controller：

```text
backend/src/main/java/com/example/sillyspringboot/auth/web/AppAuthController.java
```

前缀：

```text
/api/app/auth
```

能力：

- Telegram 登录。
- Telegram 绑定。
- H5 登录。
- H5 注册。
- H5 绑定。
- 获取当前用户。
