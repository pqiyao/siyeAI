## 文档定位

这些文档面向三类读者：

- 想快速体验项目的人：阅读快速启动和部署说明。
- 想二次开发的人：阅读项目概览、配置说明、后端说明、前端说明、接口地图。
- 想公开发布或自部署的人：重点阅读部署说明、安全检查和第三方声明。

## 重要边界

- Docker Compose 模板会构建并启动 MySQL、Redis、SillyTavern、后端、管理端和 H5 端。
- H5 端会在 Docker 镜像构建阶段自动执行 `npm run build:h5`，不需要提前提交构建产物。
- `.env`、真实密钥、数据库备份、上传文件、构建产物和本地依赖目录不应进入公开仓库。
- 当前项目使用 MIT License，详见根目录 `LICENSE`。
